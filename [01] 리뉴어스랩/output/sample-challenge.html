기획안 01
협력사 탄소 데이터 일괄 수집 자동화
1차 협력사·OEM이 수백 개 협력사에서 탄소 데이터를 수동 입력 중인 문제 해결
Priority
P0
문제 정의
1차 협력사·OEM 고객이 수백 개 협력사에서 Scope 3 탄소 데이터를 개별 수동 입력 또는 개별 링크 발송으로 수집 중. 협력사 200개 기준 주당 수십 시간 소요. CBAM 신고 기한이 다가올수록 데이터 누락 리스크 증가.
개선 제안
• 협력사 리스트 CSV 업로드 → 데이터 요청 이메일 일괄 발송
• 협력사별 응답 현황 실시간 추적 (응답률 %, 미응답 알림)
• 협력사용 간이 입력 폼 (CarbonLink 계정 불필요)

기획안 02
CBAM 신고 기한 알림 + 제출 현황 추적
OEM·1차 협력사가 신고 마감을 놓쳐 EU 세관 반려·수출 계약 위험이 발생하는 문제 해결
Priority
P1
문제 정의
OEM·1차 협력사가 CBAM 신고 마감일을 인지하지 못하거나 놓쳐 EU 세관 반려, 수출 계약 위험 등 사후 대응 케이스 반복 발생. 현재 카본링크는 보고서 생성 후 제출까지 고객이 알아서 관리.
개선 제안
• CBAM 신고 캘린더 내장: 분기별 마감일 자동 표시
• D-30 / D-7 / D-1 알림 (이메일 + 플랫폼 배너)
• 제출 체크리스트: 생성 → 검증 → 제출 3단계 상태 추적
• 반려 예방 사전 검증: EU 최신 양식 자동 대조
