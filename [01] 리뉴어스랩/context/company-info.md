# 리뉴어스랩 도메인 컨텍스트

## 기업 소개

**리뉴어스랩(Renewers Lab)** 은 공급망 탄소 데이터 공유 플랫폼 **카본링크(CarbonLink)** 를 운영하는 딥테크 스타트업입니다.

- **설립 목적**: 제조 공급망 전반의 탄소 중립 실현을 위한 데이터 인프라 구축
- **핵심 미션**: 기존 엑셀·이메일 기반의 수작업 탄소 데이터 수집·보고 프로세스를 AI 자동화로 전환
- **주요 성과**:
  - 중소벤처기업부 팁스(TIPS) 프로그램 최종 선정 (2026년 5월)
  - 액셀러레이터 김기사랩으로부터 시드 투자 유치 (2025년 12월)
  - 현대자동차, BMW, 테슬라 등 글로벌 OEM 공급망 협력사 310여 곳 고객 확보
- **대표**: 이재용 | CPO: 박태근 | COO: 김상용

## 주요 서비스

### 카본링크(CarbonLink)

공급망 탄소 배출량 측정·검증·보고 자동화 플랫폼

| 기능 | 설명 |
|------|------|
| 온실가스 배출량 측정 | Scope 3(공급망 배출량) 자동 수집 및 계산 |
| 제품탄소발자국(PCF) | 제품 단위 탄소발자국 데이터 자동화 |
| 탄소검증 자동화 | AI 기반 공급망 탄소 데이터 검증·보고 |
| 규제 대응 리포트 | CBAM, CSRD, 글로벌 고객사 보고 양식 자동 생성 |
| 공급망 데이터 공유 | 1차·2차 협력사 간 탄소 데이터 연동 |

### 컨설팅 & 교육

- 탄소배출·공급망 관리 안내 및 교육 제공
- 인천대학교 협력 ESG 세미나 진행
- 맞춤형 규제 대응 전략 수립 지원

## VoC가 중요한 이유 (비즈니스 맥락)

리뉴어스랩의 고객은 대부분 **탄소규제 대응을 처음 경험하는 실무자**들입니다. 이들은 규제 해석, 데이터 입력 방식, 신고 기한, 플랫폼 사용법 등 다양한 채널로 질문과 불만을 제기합니다.

**VoC가 비즈니스에 중요한 이유:**

1. **규제 변화 선행 지표**: 고객 문의 급증은 새로운 규제 발효나 기준 변경의 신호 — 선제적 콘텐츠·기능 대응 가능
2. **플랫폼 개선 우선순위**: 불만·요청 VoC는 카본링크 UX 개선과 기능 로드맵의 실질적 인풋
3. **이탈 위험 감지**: 처리 지연된 불만 VoC는 계약 해지 예측 변수
4. **영업 레퍼런스 발굴**: 칭찬 VoC는 세일즈 케이스 스터디와 마케팅 콘텐츠 소스
5. **규제 가이드 콘텐츠 수요 파악**: 문의 VoC 클러스터링으로 FAQ·가이드 제작 방향 설정

## 참고 용어 사전

| 용어 | 정의 |
|------|------|
| **CSRD** | Corporate Sustainability Reporting Directive. EU 기업 지속가능성 보고 지침. 일정 규모 이상의 EU 기업(및 EU에서 사업하는 외국 기업)에 ESG 정보 공시를 의무화. 2026년부터 대규모 기업(비NFRD 대상)에 확대 적용. |
| **CBAM** | Carbon Border Adjustment Mechanism. 탄소국경조정제도. EU가 탄소 집약적 수입품에 탄소 비용을 부과하는 제도. 2026년 1월 확정 시행. 초기 대상: 철강·알루미늄·시멘트·비료·전력·수소. |
| **탄소국경조정세** | CBAM의 한국어 통용 표현. EU 수출 시 제품의 탄소 내재량에 따라 CBAM 인증서 구매 비용이 발생. |
| **탄소발자국** | 제품 또는 서비스의 전 생애주기에서 발생하는 온실가스 총배출량(CO₂ 환산값). |
| **제품탄소발자국(PCF)** | Product Carbon Footprint. 특정 제품 단위의 탄소발자국. 리뉴어스랩 카본링크의 핵심 측정 지표. |
| **Scope 1** | 기업이 직접 소유·운영하는 시설에서 직접 배출하는 온실가스. |
| **Scope 2** | 기업이 구매한 전기·열·증기 생산 과정에서 간접 배출되는 온실가스. |
| **Scope 3** | 기업의 가치사슬 전반(공급망·물류·제품 사용·폐기 등)에서 발생하는 간접 배출. 리뉴어스랩의 핵심 타겟 영역. |
| **ESG** | Environmental, Social, Governance. 환경·사회·지배구조 기준의 기업 평가 체계. |
| **온실가스(GHG)** | Greenhouse Gas. CO₂, CH₄, N₂O, HFCs, PFCs, SF₆ 등 지구온난화를 유발하는 가스. |
| **탄소중립** | 온실가스 배출량과 흡수량이 동일하여 순 배출량이 0인 상태(Net Zero). |
| **NFRD** | Non-Financial Reporting Directive. CSRD 이전 EU 비재무정보 공시 지침. |
| **ETS** | Emissions Trading System. 배출권거래제. EU-ETS 탄소 허용량 거래 시장. |
