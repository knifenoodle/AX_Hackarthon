# [01] 리뉴어스랩 — VoC 분류·인사이트 리포트

당신은 리뉴어스랩의 데이터 분석 컨설턴트입니다.
이 회사의 맥락과 문제를 충분히 이해한 후 미션을 수행하세요.

## 기업 컨텍스트
→ context/company-info.md 와 context/industry-news.md 를 먼저 읽으세요.

## 미션 요약
고객사로부터 수집된 VoC CSV 데이터를 분석하여, 유형별 분류와 탄소규제 맥락 인사이트를 담은 리포트를 생성하세요.
레벨(Basic → Standard → Challenge)은 본인 수준에 맞게 선택합니다. 위 단계는 아래 단계를 포함하는 누적형입니다.

## 권장 접근 순서
1. /analyze  → data/voc.csv 파악: 컬럼 확인, 결측치 탐지, 유형 분포 확인
2. /insight  → 분류 기준 설계 + CSRD·CBAM 맥락 인사이트 도출
3. /generate → output/template.md 형식으로 결과물 생성
4. /review   → 제출 전 자가 점검

## 의사결정 로깅 규칙
분류 기준 선택, 경계 케이스 처리 방법, 인사이트 선정 근거 등 중요 결정은
반드시 decisions.md 에 기록하세요.
→ "이 결정을 decisions.md에 기록해줘" 입력 시 자동 기록됩니다.

## 채점 기준 (누적형)

### 🟢 Basic — 100점 만점 (필수)
- 분류 정확도 (40점): 4유형(불만·기능 요청·칭찬·일반 문의) 분류표 완성 + 경계 케이스 처리 근거 설명
- 건수·비율 집계 (20점): 유형별 건수와 비율(%) 정확히 산출
- 탄소규제 인사이트 (30점): CSRD·CBAM 맥락 인사이트 3개 — 단순 수치 나열 아닌 비즈니스 해석
- 제출 형식 (10점): 노션 또는 구글 독스 공개 링크로 제출

### 🟡 Standard 가산 — +30점
- 파이프라인 재현성 (15점): 새 CSV 넣어도 정제→분류→집계→리포트 동일 품질로 자동 생성
- 주간 리포트 설계 (15점): 섹션·지표·형식을 본인이 기획 + CS·제품팀 관점 근거 설명

### 🔴 Challenge 가산 — +30점
- 기획안 실효성 (20점): 제품 개선 기획안 1—2건 (문제 정의·근거 VoC·개선 제안·우선순위)
- 양식·우선순위 설계 (10점): 기획안 양식과 판단 기준을 본인이 설계 + 근거 설명

> 최소 합격선: Basic 60점 이상 (분류표 + 건수 집계 완성)
> 레벨별 만점: Basic 100 / Standard 130 / Challenge 160

## 제약 조건
- Claude Code (터미널) 또는 Cowork (채팅) 만 사용
- 별도 API 키 없음 — Claude Code 대화 안에서 직접 분류·분석 완료
- 이메일·채널톡 실시간 연동 제외 (제공된 voc.csv만 사용)
- 대시보드·웹 UI 구현 제외
- Cowork 사용 시: 배포 URL 대신 Skill(/명령) 패키징으로 제출 가능
